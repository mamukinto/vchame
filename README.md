# vchame.ge

Khinkali counter. Tap to count. Share your shame.

Vibe-coded with Claude Code in one sitting.

## Stack

- .NET 9 Minimal API + SQLite
- Vanilla JS frontend (zero dependencies)
- PWA (works offline, installable)
- Georgian + English

## Run locally

```bash
cd src/Vchame.Api
dotnet run
```
